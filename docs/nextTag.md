#### 간단 태그 정리 - 슬쩍 보고 넘어가자

##### Image

```
* next/image 의 Image 를 사용하면, img 태그를 사용하는 것보다 최적화를 해준다.
  빌드타임에는 영향이 없으며, 개발자 도구 확인히 확실히, 크기 및 용량이 줄어있었다.
```

##### CSS Modules

```
* xxxx.module.css 이란 이름으로 정해줘야 합니다.
  (className 의 유일성을 확보해준다.)

* Global Styles
  pages/_app.js
  styles/global.css
```
